# API Gravity

## Definition

API gravity is the standard measure of crude oil density, developed by the American Petroleum Institute. It is an inverted scale: higher API numbers mean lighter (less dense) oil. Light crude has API gravity above 31, medium crude falls between 22 and 31, and heavy crude is below 22. The scale is calibrated so that water has an API gravity of exactly 10. API gravity is critical because it directly determines what products a barrel of crude will yield when refined, how easily it flows through pipelines, and which refineries can process it.

## Why it matters (commodities and FX)

API gravity is the first thing a refiner or trader checks when evaluating a crude grade. Light crudes (high API) naturally yield more gasoline and diesel, the most valuable products, with less residual fuel oil. Heavy crudes (low API) yield more residuum and require expensive processing equipment (cokers, hydrocrackers) to convert bottom of the barrel into lighter products. This directly affects [[Crude Quality]] differentials and [[Crack Spread]] economics. When gasoline demand is strong, light sweet crudes command an even higher premium. API gravity also affects logistics: very heavy crudes may need to be diluted or heated for pipeline transport.

## Concrete example

[[Brent Crude]] has an API gravity of approximately 38 (light). [[Dubai Crude]] is approximately 31 (medium). Canada's Western Canadian Select (WCS) is approximately 20.5 (heavy). In a scenario where [[Brent Crude]] trades at $82/bbl, Dubai at $78, and WCS at $60, the $22 discount for WCS reflects the cost of upgrading heavy crude into useful products. If a new coking unit comes online in the US Gulf Coast, demand for heavy crude increases, and WCS narrows to a $17 discount, earning $5/bbl for a trader who was long WCS vs Brent. Conversely, if the coker goes offline, the discount widens to $27.

## Key mechanics and formulas

**API Gravity formula:**

`API Gravity = (141.5 / Specific Gravity at 60°F) - 131.5`

Where specific gravity is the density of oil relative to water (water = 1.0).

**Classification:**
- Condensate: > 45 API
- Light: 31 to 45 API
- Medium: 22 to 31 API
- Heavy: 10 to 22 API
- Extra heavy / bitumen: < 10 API

**Key benchmark API values:**
- WTI: ~39.6 API
- Brent (BFOET): ~38 API
- Dubai/Oman: ~31 API
- Mars (US Gulf): ~28 API
- Maya (Mexico): ~22 API
- Western Canadian Select: ~20.5 API

## Common misconceptions

**"Higher API always means higher price."** Very light crudes (condensates above 45 API) can actually trade at a discount because they yield too much naphtha and not enough middle distillates (diesel, jet fuel) that refiners need.

**"API gravity is the same as specific gravity."** They are inversely related. Higher API = lower specific gravity = lighter oil. The inverted scale confuses many beginners.

**"A few degrees of API difference does not matter."** Even 2 to 3 degrees of API difference can significantly change product yields and refinery processing requirements, translating to dollars per barrel in value.
