# API Gravity

## Definition

API gravity measures crude oil density on an inverted scale: higher numbers mean lighter (less dense) oil. Light crude is above 31, medium 22 to 31, heavy below 22. Water sits at exactly 10. The number determines product yield after refining, pipeline flow behavior, and which refineries can process the grade.

## Why it matters (commodities and FX)

API gravity is the first check a refiner or trader runs on a crude grade. Light crudes yield more gasoline and diesel (the most valuable products) with less residual fuel oil. Heavy crudes yield more residuum and need cokers or hydrocrackers to convert bottom of the barrel into lighter products. This drives [[Crude Quality]] differentials and [[Crack Spread]] economics. When gasoline demand is strong, light sweet grades command a higher premium. Very heavy crudes also need dilution or heating for pipeline transport.

## Concrete example

**Concrete:** [[Brent Crude]] is ~38 API (light), [[Dubai Crude]] ~31 (medium), Western Canadian Select ~20.5 (heavy). Brent at $82/bbl, Dubai at $78, WCS at $60. The $22 WCS discount prices the cost of upgrading heavy crude into useful products. A new US Gulf Coast coking unit comes online, heavy crude demand rises, and WCS narrows to a $17 discount: $5/bbl gain for a long WCS vs Brent position. If the coker goes offline, the discount widens to $27.

**Simplified:** Lighter oil makes more of the products refiners want (gasoline, diesel) without needing extra equipment. Heavier oil needs costly machinery to break down. So lighter grades sell at a premium, heavier ones at a discount. When the market builds more equipment to handle heavy oil, the discount shrinks. When that equipment breaks, the discount widens.

## Key mechanics and formulas

**API Gravity formula:**

`API Gravity = (141.5 / Specific Gravity at 60°F) - 131.5`

Where specific gravity is density relative to water (water = 1.0).

**Classification:**
- Condensate: > 45 API
- Light: 31 to 45 API
- Medium: 22 to 31 API
- Heavy: 10 to 22 API
- Extra heavy / bitumen: < 10 API

**Key benchmark API values:**
- WTI: ~39.6 API
- Brent (BFOET): ~38 API
- Dubai/Oman: ~31 API
- Mars (US Gulf): ~28 API
- Maya (Mexico): ~22 API
- Western Canadian Select: ~20.5 API

## Common misconceptions

**"Higher API always means higher price."** Very light crudes (condensates above 45 API) trade at a discount because they yield too much naphtha and not enough middle distillates.

**"API gravity is the same as specific gravity."** Inversely related. Higher API = lower specific gravity = lighter oil. The inverted scale trips up beginners.

**"A few degrees of API does not matter."** Even 2 to 3 degrees shifts product yields and refinery requirements, worth dollars per barrel.
